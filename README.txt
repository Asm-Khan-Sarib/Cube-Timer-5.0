3x3 Cube Timer 5.0
Cube Timer 5.0 is a fully responsive website created with HTML, CSS, and JavaScript. It provides a user-friendly interface for timing Rubik's Cube solves and tracking progress. The website offers various features and functionalities to enhance the solving experience.
This wesile is live now and can be used from this link: https://asm-khan-sarib.github.io/Cube-Timer-5.0/
------------------------------
Features:
1. Timer Controls: Start and stop the timer using both the Space bar and mouse clicks for convenience.
2. Auto Save: Automatically saves solving times and corresponding scrambles in the browser's local storage.
3. Adjustment Options: Easily delete or add 2 seconds to the recorded time for better accuracy.
4. View Saved Times: Access a list of all saved times along with their respective scrambles for reference.
5. Progress Report: View a summary of the solving progress.
6. Visualization: Visualize the distribution of saved times with a line chart for better analysis and comparison.
7. Time Deletion: Manually delete individual solves or quickly delete saved times.
------------------------------
The Cube Timer website utilizes the following technologies:
-> HTML: Markup language for creating the website structure.
-> CSS: Stylesheets for designing the visual appearance.
-> JavaScript: Programming language for implementing interactive features and functionalities.
